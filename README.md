1. Instalar un servidor local (Obligatorio)

La aplicación NO funcionará si se abre con doble clic.
Se necesita un servidor local.

Opción recomendada (con Visual Studio Code):

Instalar Visual Studio Code.

Instalar la extensión Live Server.

Abrir la carpeta del proyecto.

Hacer clic derecho en index.html.

Seleccionar "Open with Live Server".

2. Permitir acceso a la cámara

Cuando el navegador lo solicite, debes permitir el acceso a la cámara.

3. Imprimir los marcadores

Descargar e imprimir los marcadores:

4. Probar la aplicación

Abrir la aplicación en el navegador.

Apuntar la cámara hacia el marcador Hiro → aparecerá el modelo logomujer.glb.

Apuntar la cámara hacia el marcador Kanji → aparecerá el modelo logohombre.glb.

5. Verificar funcionamiento

Si el modelo no aparece:

Revisar que los archivos .glb estén dentro de la carpeta assets.

Verificar que el servidor esté activo.

Comprobar que la cámara tenga permisos.

-Con estos pasos la aplicación de Realidad Aumentada debería ejecutarse correctamente.
